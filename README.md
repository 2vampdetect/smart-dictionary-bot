# 📖 Умный словарь — Telegram-бот

Отправляешь боту слово — он присылает **перевод**, **пример употребления**, **синонимы** и **определение**.

Работает в обе стороны:
- английское слово → перевод на русский + определение, пример, синонимы;
- русское слово → перевод на английский + определение, пример и синонимы для перевода.

Используются **бесплатные API без ключей**:
- [MyMemory](https://mymemory.translated.net) — перевод
- [Free Dictionary API](https://dictionaryapi.dev) — определения и примеры
- [Datamuse](https://www.datamuse.com/api/) — синонимы

---

## 🚀 Быстрый старт

### 1. Создайте бота в Telegram
1. Откройте в Telegram [@BotFather](https://t.me/BotFather).
2. Отправьте команду `/newbot`.
3. Придумайте имя (например, «Умный словарь») и username (например, `my_smart_dict_bot`).
4. BotFather пришлёт **токен** вида `123456789:AAE0abc...` — скопируйте его.

### 2. Установите зависимости
Нужен Python 3.10+.

```bash
cd smart-dictionary-bot
python -m venv venv

# Linux / macOS:
source venv/bin/activate
# Windows:
# venv\Scripts\activate

pip install -r requirements.txt
```

### 3. Запустите бота

```bash
# Linux / macOS:
export BOT_TOKEN="123456789:AAE0abc..."
python bot.py

# Windows (PowerShell):
# $env:BOT_TOKEN="123456789:AAE0abc..."
# python bot.py
```

Готово! Откройте своего бота в Telegram, нажмите **Start** и отправьте любое слово, например `serendipity` или `вдохновение`.

## ☁️ Как запустить 24/7 (хостинг)

Пока скрипт запущен на вашем компьютере — бот работает. Чтобы он работал постоянно, разместите его на сервере:

- **PythonAnywhere / Railway / Render** — простые варианты для новичков (есть бесплатные тарифы).
- **Любой VPS** — запустите через `systemd` или в `tmux`/`screen`.

На сервере достаточно тех же трёх шагов: скопировать файлы, `pip install -r requirements.txt`, задать `BOT_TOKEN` и запустить `python bot.py`.

## ⚙️ Как это устроено

- `bot.py` — весь код бота (aiogram 3, long polling, вебхуки не нужны).
- Язык слова определяется автоматически по наличию кириллицы.
- Все запросы к API асинхронные, с таймаутом 10 секунд; при сбое одного источника бот отдаёт то, что удалось найти.

## 🛠 Идеи для доработки

- Кнопка «🔊 Озвучить» (Text-to-Speech).
- История запросов и личный словарик пользователя (SQLite).
- Карточки для повторения слов (интервальное повторение).
- Русские синонимы через словарь синонимов или LLM API.
